The basic idea:

Add Entries

Entries are things that you want to keep track of. They are really just one or more data points, stored as key:value, or just key.

Example: headache

Example: wake energy:2

Example: weight:150

Example: running shoes:new-balance miles:14

Create Stats

Stats are how you define and filter the data you want to study. There are currently four main types of stats: average, usual, sum, and count.

Example: average weight 1h after waking

Example: sum miles (shoes:new-balance)

Example: usual shoes

Example: count (daily) headache
